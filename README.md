# Taiwan-CWB-Data
Taiwan Central Weather Bureau Data

* Web system Demo  Beta v1:
https://yichunsung.shinyapps.io/TaiwanCWB/

## 資料來源

所有資料來源都是台灣中央氣象局資料

* 台灣中央氣象局網站：
http://www.cwb.gov.tw/eng/index.htm
* 中央氣象局資料查詢系統：
http://e-service.cwb.gov.tw/HistoryDataQuery/index.jsp
